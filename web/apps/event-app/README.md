# @furria/event-app — Stub

The guest-facing event web app (no install): per-table photo upload, live photo wall and the digital programme during events. Planned stack: React 19 + TypeScript + Vite + MUI + TanStack Router/Query, themed exclusively via `@furria/ui` — a plain responsive web app reached via QR code, no Capacitor wrapper planned. Design system and context live in [`docs/design/`](../../../docs/design/README.md). Not scaffolded yet — this directory intentionally contains no `package.json`.
